= Holidays

A simple Java library to deal with holidays. The library uses as backend the excellent Ruby Holidays Gem
available on https://github.com/alexdunae/holidays

== Usage

=== Examples

Have a look at the two test classes ch.gadp.holidays.RegionsTest and ch.gadp.holidays.HolidaysTest

=== Usage
See javadocs in the apidocs folder
